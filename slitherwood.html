Parameters
Min region size 3
Stop splitting when both w and h ≤ this — higher = fewer, larger regions
Split bias 50%
Probability of a horizontal split — 0% = always vertical, 100% = always horizontal
Aspect ratio threshold 1.5
Force split direction when one side is this many times longer — lower = more forced splits
Split margin 2
Tiles excluded from edges when placing a split — higher = more centered, more equal-sized regions
Hallway threshold 1
A region is a hallway if its narrowest side is ≤ this — raise it to classify wider corridors as hallways
Door count 1
Openings punched through each split line — higher = more connectivity and loops
Stop condition
AND: stop when both dimensions ≤ min. OR: stop when either dimension ≤ min (prevents long thin rooms)
Seed
Sliders use the current seed — Regenerate and 🎲 pick a new one
